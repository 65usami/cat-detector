# cat-detector

## Feature

Show possibility of your picture whether a cat by AI.

## Demo

http://usa-mi.work/

## System

- Ubuntu 16.04 LTS
- Python: 3.6.7
- Docker: 19.03.2
    - docker-compose: 3.7.3
- Nginx + uWSGI + Flask
- TensorFlow + Keras

## Usage
```
$ docker-compose up -d
```
